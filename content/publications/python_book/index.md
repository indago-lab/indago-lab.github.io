{{< /admonition >}}

This book provides a comprehensive introduction to Python programming for engineering and technical studies. It covers Python fundamentals as well as advanced topics such as numerical methods, data analysis, visualization, optimization, and differential equations using libraries like NumPy, SciPy, and Matplotlib. Written with a strong practical focus, the book combines theory with numerous examples relevant to engineering applications. It is intended as a free and open textbook for students across all courses where Python is used as a computational and problem-solving tool.

Courses covered in the textbook:
- Computer Applications in Engineering (complete)
- Numerical methods (complete)
- Computational engineering (partial)
- Modelling in Engineering (partial)
- Optimization in Engineering (partial)
- Systems and Data Analysis (partial)
